## Progression Guide

### Recommended Player Levels
- **Levels 1-5**: Player Level 5-10
- **Levels 6-10**: Player Level 10-15
- **Levels 11-15**: Player Level 15-25
- **Levels 16-20**: Player Level 25-50

### Resource Progression
Each tier introduces new materials:
- **Tier 1**: Iron, Steel, Titanium
- **Tier 2**: Vanadium, Tungsten, Osmium
- **Tier 3**: Iridium, Chromium, Mythril
- **Tier 4**: Adamantite, Orichalcum

### Strategic Tips

1. **Don't Rush**: Each level requires defeating all monsters before descending
2. **Resource Management**: Stock up on health items between levels
3. **Level Appropriately**: Match your player level to the dungeon tier
4. **Equipment Matters**: Upgrade weapons using rare materials from deeper levels
5. **Learn Patterns**: Each monster type has specific strengths
6. **Save Points**: Return to town to deposit treasures periodically
7. **Final Boss Prep**: Level 20 requires extensive preparation
